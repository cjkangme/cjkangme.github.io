# EP 05. 시맨틱(Semantic) 태그

### 시맨틱 태그 : '의미론적인 태그'로, 기본적으로 `<div>`와 동일한 역할을 수행하지만 태그에 의미를 부여함으로써 각 태그의 의미를 보다 쉽게 알 수 있도록 한다.

<br>

## 1. 시맨틱 태그의 장점

1. 검색엔진 최적화
   (검색 엔진에게 어떤 태그가 어떤 태그를 갖는지 알려주어 데이터를 효율적으로 추출하도록 함)
2. 웹 접근성 향상 (`<em>` 및 `<strong>`과 비슷한 예)
3. 코드의 가독성 향상

<br>

## 2. 시맨틱 태그의 종류

![layout](/images/2022-09-11-first/layout.png)

<br>

- `<header>` : 페이지 최상단에서 페이지에 대한 소개, 탐색을 돕는 컨텐츠를 나타내는 영역
- `<nav>` : 현재 페이지 내 또는 다른페이지로의 링크를 보여주는 영역
- `<main>` : HTML body의 주 콘텐츠를 나타내는 영역
- `<aside>` : 페이지의 주요내용과 간접적으로만 관련이 있어, 따로 분리된 내용을 나타내는 영역
- `<section>` : 문서나 응용프로그램 하나가 일반적으로 차지하는 하나의 구획
- `<article>` : 여러가지 아이템들을 묶어서 재사용 가능하게 그룹화한 영역
- `<footer>` : 페이지 최하단에서 현재 웹사이트에 대한 정보를 표시하는 영역

<br>

보다 많은 시맨틱 태그 종류나, 자세한 정보를 얻으려면 MDN Semantics의 용어사전 참고
<br> **[MDN Semantics 용어사전 링크](https://developer.mozilla.org/ko/docs/Glossary/Semantics)**

# EP 06. Block vs. Inline Element

## 1. div 태그와 span 태그

- `<div>` : 박스형태로 하나 이상의 태그를 담아 각 영역을 분할하는 일종의 컨테이너 역할을 수행 (콘텐츠 크기와 관련 없이 한 블럭을 차지)
- `<span>` : 특정 텍스트, 이미지와 같은 하나의 아이템을 담는 역할 수행 (정확히 콘텐츠 만큼의 공간을 차지)

<br> div와 span의 차이는 'display' 속성의 기본값이 다르기 때문이다.

- `<div>` - display: block (Block Level Element)
- `<span>` - display: inline (Inline Level Element)

<br>

## 2. Block vs. Inline

1. 블록 레벨 요소(Block Level Element)
   - 블록 레벨 요소는 부모 요소의 공간을 차지하여 '블록'을 만든다.
   - 블록 레벨 요소는 한 칸을 모두 차지하기 때문에 크기와 상관없이 세로로 나열된다. (향후 배울 display 속성을 통해 변경할 수 있음)
   - width, height, margin 등의 속성을 통해 크기 및 콘텐츠간 여백을 자유롭게 조절할 수 있다.
   - 블록 레벨 요소의 예시
     - `<h1>` ~ `<h6>`
     - `<ol>`, `<ul>`, `<li>`
     - `<p>`
       등

<br>

2. 인라인 레벨 요소(Inline Level Element)
   - 인라인 레벨 요소는 줄바꿈 없이 요소를 구성하는 태그에 할당된 공간만을 차지한다.
   - 인라인 레벨 요소는 공간이 허락하는 한 가로로 나열된다.
   - width, height, margin 등의 조절이 불가능하다. (콘텐츠간 조절도 불가능해 line-height을 이용해야 함)
   - 내부 콘텐츠를 정렬하는 속성 역시 사용할 수 없다.
   - 인라인 레벨 요소의 예시
     - `<a>`
     - `<em>`
     - `<img>`
     - `<span>`
       등

<br>

## 3. 예시

- 코드

```HTML
<head>
    <style>
      div {
        background: yellowgreen;
        width: 100px;
        height: 100px;
        border: 1px solid black;
      }

      span {
        background: skyblue;
        width: 100px;
        height: 100px;
        border: 1px solid black;
      }
    </style>
  </head>
  <body>
    <br /><br />

    <div>div1</div>
    <div>div2</div>
    <div>div3</div>

    <span>span1</span>
    <span>span2</span>
    <span>span3</span>
  </body>
```
