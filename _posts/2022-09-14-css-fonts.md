# 폰트 관련 속성

## 1. font-family

글꼴의 종류를 지정하는 속성이다.
<br> 콤마로 구분하여 여러가지 폰트를 지정할 수 있다.

- 가장 첫번째로 선언된 폰트가 우선 적용된다.
- 해당 폰트를 사용할 수 없을 경우, 다음으로 선언된 폰트가 적용되는 식으로 폰트가 선택된다.

### 웹 폰트

사용자가 컴퓨터에 설치하지 않아도, 온라인의 특정 서버에 위치한 파일을 불러와 사용할 수 있도록 한 웹 전용 폰트를 말한다.
<br> 대표적으로 [Google Fonts](https://fonts.google.com/) 가 있다.

- 구글 폰트 적용법

1. 원하는 폰트와 스타일을 찾아 선택한 뒤, 우측 상단의 'View selected families' 아이콘을 클릭한다.
2. @import 본문은 CSS 문서 또는 style 태그에 선언한다.
3. 글꼴을 적용하고 싶은 요소에 font-family: '글꼴명'을 선언한다.

```css
/* 나눔고딕 폰트 등록 */
@import url("https://fonts.googleapis.com/css2?family=Nanum+Gothic:wght@400;700;800&family=Noto+Sans+KR&display=swap");

/* html 문서 전체에 해당 글꼴 적용 */
html {
  font-family: "Nanum Gothic", sans-serif;
  font-size: 16px;
}
```

## 2. font-size, font-weight

font-size는 CSS의 다양한 단위를 사용하여 글꼴의 크기를 변경한다.<br>
font-weight는 마찬가지로 단위를 사용하여 글꼴의 굵기를 변경한다.

- px : 가장 기본적인 픽셀 단위
- rem : 최상위 문서(html)에 정의된 font-size 값을 기준으로 n배를 적용하는 상대 단위 (rem에서 r이 root를 뜻함)
  - 최상위 문서에 별도 정의가 없을 경우 1rem = 16px가 기본값이다.

```css
html {
  font-size: 16px;
}

.subtitle-rem {
  font-size: 2rem; /* 32px */
}
```

- em : 해당 단위가 사용되고 있는 요소의 font-size를 기준으로 n배를 적용하는 상대 단위

```css
.article {
  font-size: 10px;
}

.artice > div {
  font-size: 1.5em; /* 15px */
}
```

- vw : viewport width란 뜻으로 현재 브라우저의 가로 너비를 기준으로 퍼센트를 적용하는 상대 단위

- vh : viewport height란 뜻으로 현재 브라우저의 세로 너비를 기준으로 퍼센트를 적용하는 상대 단위

> 기타 단위는 [MDN](https://developer.mozilla.org/ko/docs/Learn/CSS/Building_blocks/Values_and_units)을 참고

## 3. color

글자의 색상을 지정한다.<br>
총 3가지 방법으로 색상을 표현할 수 있다.

1. 키워드 표기법

- CSS에 기본적으로 정의되어있는 키워드를 이용하여 특정 색을 나타낸다.
- tranparent : 완전히 투명색을 의미하며 rgba(0,0,0,0)과 동일하다.

2. RGB/RGBA 표기법

- rgb : RGB코드를 통해 특정 색을 표현한다.
- rgba : rgb에 더해 투명한 정도까지 표현한다. (0이 완전 투명, 100이 완전 불투명)

3. 16진수(HEX color code) 표기법

- '#' : 16진수 코드를 통해 특정 색을 표현한다. 표현 방법이 다를 뿐 기본적으로 rgb/rgba와 동일하다.

```css
/* 모두 동일한 색상을 나타낸다. */

/* 키워드 표기법 */
.title {
  color: yellow;
}

/* RGB/RGBA 표기법 */
.title {
  color: rgb(255, 255, 0);
  color: rgba(255, 255, 0, 100);
}

/* 16진수 표기법 */
.title {
  color: #ffff00;
}
```

## 4. 기타 관련 속성

> 자세한 사용법은 MDN을 참고하는 것이 좋다.

- font-style : 이탤릭체 등 글자의 스타일을 지정한다.
- text-align : 요소 내에서 글자의 정렬 방향을 표시한다.
  - left, center, right, justify(양쪽 맞춤) 지정 등이 가능하다.
- line-height : 한 줄이 차지할 높이를 설정하는 것으로, 줄 간격 설정할 수 있다.
  - 현재 font-size에 비례해야하므로 rem, em과 같은 상대 단위를 사용하는 것이 권장된다.
  - 텍스트의 세로 정렬에도 사용할 수 있다. (요소의 세로 크기와 line-height값이 같을 때 텍스트가 가운데 오는 것을 이용)
- letter-spacing : 글자의 자간을 조절한다.
- word-spacing : 단어 사이의 간격을 조절한다. (띄어쓰기 간격과 유사)
- text-indent : 첫줄의 들여쓰기 간격을 조절한다.
- text-transform : 소문자-대문자 변경
- text-decoration : 글자의 취소선 및 밑줄 등을 설정한다.
  - color, line, style, thickness 등의 하위 속성이 있다.

```css
/* text-decoration 예시 */

.article {
  font-size: 30px;
  text-decoration: line-through dotted 3px purple;
}
```
