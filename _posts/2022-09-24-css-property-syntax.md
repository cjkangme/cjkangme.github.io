![CSS 속성 syntax](/assets/images/github/css-syntax.png)

Visual Studio Code는 사용할 CSS 속성에 마우스를 올리면, 위와 같이 친절하게 속성에 대한 설명과 속성값 구문, 그리고 참조 링크를 제공해준다.<br>
앞으로 내가 알아야 할 속성이 많을텐데, 이참에 속성값 구문을 읽는 법을 배워두면 좋을 것 같아서 한 번 정리해보려고 한다.

# Values type (유효한 값 형식)

1. keyword(키워드): 사전에 정의된 의미가 있는 값을 말한다.
   - 별도 기호 없이 그대로 쓴다. 또한, CSS 문서에서 대소문자를 구분하지 않는다.
   - 예시. `auto`, `thin`, `thick` 등
2. basic data types(기본 데이터 타입): CSS 전반에서 사용하는 길이, 색상값 등의 중요한 자료형을 말한다.
   - 따옴표 없이 화살 괄호(`<`, `>`)로 감싼다.
   - 예시. `<color>`, `<length>`, `<integer>`
   - 범위가 존재하는 숫자 데이터 타입의 경우 대괄호(`[`, `]`를 사용하여 범위를 나타낼 수 있다.
   - 예시. `<integer [0,10]>`
3. 속성 데이터 타입 : 값을 참조하는 속성의 이름을 나타낸다. 이 데이터 타입값을 그대로 속성으로 사용할 수도 있다. 주로 단축 속성 구문에서 많이 볼 수 있다.
   - 작은 따옴표(`'`)와 화살 괄호(`<`, `>`)로 감싼다.
   - 예시. `<'font-weight'>`, `<'border-style'>`
4. 비 속성 데이터 타입 : 기본 데이터 타입과 유사하다. 기본 데이터 타입은 CSS 전반에서 사용되지만, 비 속성 데이터 타입은 이를 사용하는 속성과 가까운 곳에 정의되어 있다는 차이가 존재한다.
   - 따옴표 없이 화살 괄호(`<`, `>`)로 감싼다.
   - 예시. `<line-width>`

# Component Value Combinators (구성요소 값 결합 기호)

해당 구성요소 값이 속성에 필수인지 아닌지, 순서가 어떻게 되는지를 나타내는 기호이다.

1. 공백(` `): 모든 개체가 필수, 순서 유지 필수
2. 이중 앰퍼샌드(`&&`): 모든 개체가 필수, 순서는 변경 가능
3. 이중 바(`||`): 모든 개체 중 하나 이상이 필수, 순서는 변경 가능 (단일 바보다 우선순위 높음)
4. 단일 바(`|`): 모든 개체 중 하나만 선언 가능
5. 대괄호(`[]`): 그룹을 나타냄

# Component Value Multipliers (구성요소 값 증가 기호)

해당 구성요소 값을 몇 번이나 반복할 수 있는지를 나타내는 기호이다. 구성요소 값 기호가 없다면 한 번만 선언해야 한다.

1. 별표(`*`): 횟수 제한 없음
2. 더하기(`+`): 1회 이상 선언 필수, 횟수 제한 없음
3. 물음표(`?`): 반드시 0회 또는 1회 선언 가능
4. 중괄호(`{}`): 선언 가능 범위를 나타냄
   - {A}: 반드시 A회 선언
   - {A, B}: 최소 A회, 최대 B회 선언
   - {A, }: 최소 A회 선언, A회 이상은 횟수 제한 없음
5. 해시(`#`): 더하기와 똑같이 1회 이상 선언 필수, 서로 쉼표를 통해 구분해야함
6. 느낌표(`!`): 그룹을 나타내는 기호인 대괄호 뒤에 사용하며, 그룹내에서 적어도 1회 선언되야 한다는 것을 표시

# 예시

![CSS 속성 syntax](/assets/images/github/css-syntax.png)

`[ <'font-style'> || <font-variant-css2> || <'font-weight'> || <font-stretch-css3> ]?`: 해당 그룹 내에서 하나만 선언하거나, 선언하지 않을 수 있다.<br>
`<'font-size'>`: 공백으로 구분되어 있으므로, 필수로 선언해야 한다.<br>
`[ / <'line-height'> ]?`: 0회 또는 1회 선언할 수 있다. (`/`은 값 구분을 위해 사용해야하는 구분자이다)<br>
`<'font-family'>`: 공백으로 구분되어 있으므로, 필수로 선언해야 한다.
`caption | icon | menu | message-box | small-caption | status-bar`: 그룹으로 묶인 값들을 대신할 수 있는 미리 정의된 값들이다. 해당 값중 하나만 선언할 수 있다.

<br> 다음은 선언할 수 있는 유효한 값의 예시이다.

```css
font: 16px sans-serif;
font: italic 20px sans-serif;
font: bold 16px /50px sans-serif;
font: menu;
```
